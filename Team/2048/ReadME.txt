2048
. The famous 2048 game written in JavaScript
. An addictive game with great fun.
. This game is developed using Html , CSS and JavaScript.

.How to play the game:
  + Open the index.html file and choose the game or directly open 2048.html in the 2048 folder.
  + Press start.
  + Play the game by pressing the "A, W, D, S" keys on the keyboard.
  + Try to add move similar squares together to create on square with a duplicate value.
  + Try to reach the ultimate goal of a 2048 square with reaching a deadend.
